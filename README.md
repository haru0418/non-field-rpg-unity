## アプリケーション名
- non-field-rpg-unity


## 概要
- ノンフィールドゲーム
- ターン制のバトル仕様

## 本番環境
- 設定していません

  
## 制作背景（意図）
- ゲーム開発がどういうものか知ってみたくて、Unityを使って作成しました。<br>
C#を使ったことがなかったですがUdemyのサイトを使ってやってみました。
